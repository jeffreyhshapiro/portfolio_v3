GYMtrition

Project Description

GYMtrition is a total fitness gym solution web app that helps you locate a gym near you, lets you build a workout plan, and then make a healthy post workout meal. It was the front-end project for the Rutgers Coding Bootcamp course.

I developed the workout plan module by sourcing data about exercises from all over the internet and organizing it into JSON. This gave the module an 'API' full of exercise data for the user to populate into a dynamic table constructed with jQuery. Curious to see how it works? Check it out!

Technologies Used

  • HTML5
  • CSS3
  • jQuery
  • MaterializeCSS

Tone Down For What

Project Description

Tone Down For What is a service that helps you to figure out how your email sounds when you're not too sure. It uses IBM Watson's tone analyzer API to give you several metrics about your text, including how angry, joyous, sad and analytical (and more!) you sound. It was the back-end project for the Rutgers Coding Bootcamp course.

I enabled the Watson API to render data to Highcharts through a series of Angular controllers. I also enabled this single page web app to perform all CRUD actions. I enjoyed interacting with MongoDB via Angular in this project.

Technologies Used

  • AngularJS
  • MongoDB
  • Express
  • Bootstrap
  • HTML5
  • CSS3
